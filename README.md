# final-course-server
## Deploy at 
       https://finall-task.onrender.com/

### Installing 
       npm i

### Run server 
       npm run dev 



# API

## Authentification 

### Registration 
       api/auth/registration
       request body is:
      username, password, role (all string)
### Login 
       api/auth/login
     request body is:
     username and password
### Chech (update authentication token)
       api/auth/check
      req.body is : 
      user
### Get All Users
       api/auth/users
